PRE PROJETO

No Pré Projeto nós fizemos uma rede em grupo com, 50 computadores, 5 impressoras e com 20% dos dispositivos tento que ser moveis, de uma rede de ensino da Proz dividindo tudo em quatro andares. Nesse Pré Projeto nós também aprendemos à configurar um servidor para ele ser Static ou DHCP, configurando ele e o Ip dele para distribuir aos outros computadores, para assim não termos o trabalho de configurar o Ip um por um. E aprendemos também como configuarar a rede Wi-Fi.
